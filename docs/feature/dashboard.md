# Dashboard

Dashboard merupakan pusat kontrol yang menyajikan gambaran lengkap performa bisnis Anda secara real-time. Halaman ini membantu Anda memantau penjualan, transaksi, produk, pelanggan, dan aktivitas bisnis secara cepat dalam satu tampilan.

![Dashboard Overview](/dashboard-overview.png)

---

## Ringkasan Statistik

Dashboard menampilkan empat metrik utama yang selalu diperbarui secara otomatis.

| Statistik          | Deskripsi                  | Indikator                              |
| ------------------ | -------------------------- | -------------------------------------- |
| Total Revenue      | Pendapatan keseluruhan     | Tren naik/turun dengan sparkline chart |
| Total Transactions | Jumlah transaksi selesai   | Perubahan persentase periodik          |
| Total Products     | Jumlah produk aktif        | Indikator stok dan perubahan           |
| Total Customers    | Jumlah pelanggan terdaftar | Pertumbuhan pelanggan                  |

![Statistics Cards](/statistik-overview.png)

## Sales Analytics

### Filter Periode

- Weekly (Mingguan)
- Monthly (Bulanan)
- Yearly (Tahunan)

## Refresh Data

### Auto Refresh

Data dashboard diperbarui secara otomatis secara berkala untuk memastikan informasi selalu terbaru.

### Manual Refresh

Gunakan tombol **Refresh** atau tekan shortcut `R` untuk memperbarui data secara manual.

::: tip Tips Penggunaan

- Pantau grafik penjualan setiap hari untuk mendeteksi tren lebih awal.
- Periksa notifikasi stok secara rutin agar tidak kehabisan barang.
- Gunakan filter periode untuk analisis data yang lebih spesifik.
- Lakukan refresh jika data tampak belum terbaru.
- Backup database secara berkala untuk keamanan data.
  :::
